## What is Groupchat?

Groupchat is a terminal-native messaging platform designed for developers who want to stay in their workflow. Built on OpenTUI, it brings real-time team communication directly into your command line—with embedded AI agents, live presence, and a keyboard-driven interface.

## Using Claude in Groupchat

In any channel, invoke Claude as an embedded agent using the `/claude` command. Claude runs inline with full tool use capabilities—it can read files, execute commands, and provide answers without requiring you to switch applications.

- Windows (x64)
