## Overview

The `/claude` command enters Claude Code agent mode, allowing you to interact with Claude AI directly within the chat interface. Use `/exit` to leave agent mode.

## /claude Command

Enter Claude Code agent mode.

### Syntax

```bash
/claude
```

### Parameters

No parameters required.

## Agent Mode System

The agent system uses local event handling:

### Event Detection

The system provides helpers to detect agent-related events:

```typescript
// Check if event enters an agent
isAgentEnterCommandEvent(eventType: string): boolean

// Check if event exits an agent
isAgentExitCommandEvent(eventType: string): boolean

// Check if event is any agent command
isAgentCommandEvent(eventType: string): boolean
```

Source: `src/lib/commands.ts:242-255`

### Extract Agent ID

Get the agent ID from an enter event:

```typescript
parseAgentIdFromEnterEvent(eventType: string): string | null

// Example:
```

## Agent Interaction Flow

<Steps>
  <Step title="Enter Agent Mode">
    User types `/claude` and presses Enter
    
    System sends `local_agent_enter:claude-code` event
  </Step>
  
  <Step title="Agent Active">
    All subsequent messages route to Claude
    
    UI indicates agent mode is active
  </Step>
  
  <Step title="Exit Agent Mode">
    User types `/exit` and presses Enter
    
    System sends `local_agent_exit` event
  </Step>
  
  <Step title="Normal Chat Resumed">
    Messages return to normal channel behavior
  </Step>
</Steps>
