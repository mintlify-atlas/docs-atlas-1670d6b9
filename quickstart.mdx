## Quick Start Guide

This guide will walk you through installing Groupchat, authenticating your account, joining a channel, and sending your first message—all from your terminal.

<Steps>
  <Step title="Install Groupchat">
    Install Groupchat globally using your preferred package manager:

    <CodeGroup>
      ```bash npm
      npm install -g groupchat
      ```

      ```bash yarn
      yarn global add groupchat
      ```

      ```bash pnpm
      pnpm add -g groupchat
      ```

      ```bash bun
      bun add -g groupchat
      ```
    </CodeGroup>

    <Tip>
      Make sure you have Node.js 18+ or Bun 1.0+ installed on your system.
    </Tip>
  </Step>

  <Step title="Launch Groupchat">
    Run the `groupchat` command to launch the application:

    ```bash
    groupchat
    ```

    On first launch, the terminal will clear and display the login screen.
  </Step>

  <Step title="Authenticate via browser">
    Since this is your first time running Groupchat, you'll need to authenticate:

    1. The app will automatically open your browser to `https://app.groupchatty.com/auth/cli`
    2. A local authentication server will start on `http://localhost:8080` (or another port in the 8080-8099 range)
    3. Complete the authentication flow in your browser
    4. Once authenticated, you'll see a success message in the browser
    5. Return to your terminal—you're now logged in!

    <Note>
      Your authentication token is securely stored in your system's keychain and will persist across sessions.
    </Note>
  </Step>

  <Step title="Navigate to channels">
    After authentication, you'll see the main menu. The menu displays:
    - Your available channels
    - Direct messages
    - Unread message counts
    - Online presence indicators

    Use keyboard shortcuts to navigate:
    - **Arrow keys** or **j/k** to move through the list
    - **Enter** to select a channel
    - **ESC** to return to the menu from any screen
  </Step>

  <Step title="Join or create a channel">
    To join an existing channel:
    1. Navigate to the channel in the menu
    2. Press **Enter** to open it

    To create a new channel:
    1. Press **c** from the menu (or navigate to "Create Channel")
    2. Enter the channel name
    3. Press **Enter** to create and join
  </Step>

  <Step title="Send your first message">
    Once in a channel:
    1. Type your message in the input field at the bottom
    2. Press **Enter** to send
    3. Your message appears instantly with real-time delivery

    ```
    [10:24 AM] you: Hello from the terminal!
    ```

    <Tip>
      Messages support markdown formatting, code blocks, and inline code.
    </Tip>
  </Step>

  <Step title="Try the Claude agent">
    Invoke Claude in any channel using the `/claude` command:

    ```
    /claude what is the capital of France?
    ```

    Claude will respond inline with full context from the conversation. You can also give Claude more complex tasks:

    ```
    /claude analyze this error and suggest a fix: [paste your error]
    ```

    Claude has access to:
    - Read files in your project
    - Execute commands
    - Search documentation
    - Answer questions with full context

    <Note>
      Claude runs with tool use capabilities, meaning it can perform actions beyond just answering questions.
    </Note>
  </Step>
</Steps>

## Essential Keyboard Shortcuts

Here are the most important shortcuts to get you started:

| Shortcut | Action |
|----------|--------|
| **ESC** | Return to menu |
| **Ctrl+C** | Exit (press twice) |
| **Ctrl+O** | Logout |
| **Ctrl+E** | Toggle user list in channels |
| **↑/↓** or **j/k** | Navigate lists |
| **Enter** | Select/Send |
| **c** | Create channel (from menu) |

## Troubleshooting

### Browser doesn't open automatically

If the browser doesn't open during authentication, manually visit:

```
https://app.groupchatty.com/auth/cli?state=<state>&redirect_uri=http://localhost:8080/callback
```

The state parameter and redirect URI will be displayed in your terminal.

### Authentication times out

The authentication server times out after 5 minutes. If you see a timeout error:
1. Exit Groupchat (**Ctrl+C** twice)
2. Run `groupchat` again to restart the auth flow

### Can't exit the application

Press **Ctrl+C** twice within 3 seconds to exit Groupchat. You'll see a status message after the first press:

```
Press Ctrl+C again to exit
```
